# -# 🏋️‍♂️ Training Planner: Планировщик тренировок

**Автор:** [Ванюшкин Данил]
**Язык:** Python 3.x

## Описание проекта

Графическое приложение (*Desktop App*) для ведения личного дневника тренировок. Позволяет добавлять новые занятия с указанием даты (ГГГГ-ММ-ДД), типа и длительности.

**Ключевые особенности:**
*   **Сохранение данных:** Вся информация сохраняется в файл `trainings.json` при закрытии программы или добавлении новой записи.
*   **Фильтрация:** Удобный поиск по типу тренировки («Все», «Кардио», «Силовая» и др.) и по конкретной дате.
*   **Валидация:** Проверка корректности ввода дат и числовых значений длительности.
*   **Простой интерфейс:** Интуитивно понятное управление с помощью стандартных элементов *Tkinter*.

## Технологии

*   **Python 3.x** — язык программирования.
*   **Tkinter** — стандартная библиотека *Python* для создания *GUI*.
*   **JSON** — формат хранения структурированных данных.
*   **Git** — система контроля версий.

## Как запустить

> Убедитесь, что у вас установлен Python версии 3.6 или выше.
> Для работы программы не требуются сторонние библиотеки (кроме стандартных).

1.  Скачайте или клонируйте репозиторий.
2.  Перейдите в папку проекта через терминал или командную строку.
3.  Запустите приложение командой:
    ```bash
    python main.py
    ```
4.  Используйте интерфейс для добавления тренировок. Данные сохраняются автоматически.
